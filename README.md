这是一个使用androidstudio编写的双语登陆，他的软件交互有一个账号输入框，密码输入框，确定按钮和设置按钮。登陆页面背景以蓝色为主色调，设置语言界面以粉紫色为主色调。设置页面有自动（根据系统语言，确定自己的语言），中文和英文三种设置。
在实现登陆功能方面，确定按钮的监听器感应到点击确定按钮后，会匹配密码和账号，两个都匹配成功了后，会跳出登陆成功和帐户或密码提示信息。在中英文切换实现方面，通过选择的选择中文或英文，分别选用string.xml(cn)和string.xml(en)上的设置,实现页面中英文的转换。
